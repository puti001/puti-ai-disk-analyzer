# Puti-AI 磁碟空間分析工具 (Puti-AI Disk Space Analyzer)

Puti-AI 磁碟空間分析工具是一款專為 Windows 環境設計的**淺色毛玻璃 (Glassmorphism) 風格**本機磁碟與資料夾空間分析軟體。透過本工具，您可以輕鬆掃描、視覺化檢視硬碟空間的佔用狀況，並快速找出佔用空間的元兇！

## 🌟 特色功能

1. **極致美觀的視覺介面**：採用現代淺色 Glassmorphism 毛玻璃視覺風格，搭配流暢的 Chart.js 檔案類型佔比環狀圖。
2. **零命令列小黑窗啟動**：主程式直接編譯為 Windows GUI 子系統，雙擊主程式即可在背景靜默啟動 Node.js 服務，完全不彈出干擾的小黑視窗，不再需要額外的 VBS 啟動器。
3. **網頁一鍵安全關閉**：點擊網頁右上角的「關閉工具」按鈕，背景服務會自動關閉並釋放系統資源。
4. **本機安全與零依賴**：完全在您的本機電腦執行，不需要安裝額外環境，保障您的個人隱私與資料安全。
5. **跨磁碟瀏覽器**：支援 C 槽、D 槽、OneDrive 與其他外部雲端/虛擬磁碟的無縫切換，單擊選取、雙擊深入。
6. **歷史導覽控制**：具備 ⬅上一頁、➡下一頁、⬆往上一層、🏠回到磁碟根目錄的網頁歷史紀錄導覽功能。
7. **十大巨無霸檔案定位**：自動為您篩選出該目錄下體積前 10 大的超大檔案，**支援一鍵自動開啟 Windows 檔案總管並高亮選取該檔案**，方便您快速清理空間！

## 🚀 快速開始

這是一個免安裝的綠色軟體，請按照以下步驟執行：

1. 下載並解壓縮本專案。
2. 雙擊執行 **`puti-ai-disk-analyzer.exe`**。
3. 系統將會自動在背景啟動服務，並自動在瀏覽器中打開工具網頁（網址為 `http://localhost:3000`）。
4. 使用完畢後，點擊網頁右上角「**關閉工具**」即可結束背景執行。

## 📂 專案檔案結構說明

* `puti-ai-disk-analyzer.exe`：打包好的 Windows 獨立執行檔 (Node.js 後端服務，內建 Windows GUI 子系統以隱藏小黑窗)。
* `Puti-AI-使用說明.md` / `README.md`：使用指南與專案說明。
* `server.js`：後端 Node.js 原生 API 伺服器與快速掃描引擎。
* `public/`：前端網頁資源。
  * `index.html`：主視覺介面。
  * `style.css`：毛玻璃視覺與排版設計。
  * `app.js`：前端邏輯、歷史狀態控制與 SSE 即時進度處理。
  * `instructions.html`：網頁版使用說明書。
